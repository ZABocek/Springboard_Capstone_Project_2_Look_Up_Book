## 📝 Quick Reference

### Essential Commands

**Setup:**
```bash
cd server
npm install
```

**Load Database:**
```bash
psql -U postgres -d look_up_book_db -f consolidated_database.sql
```

**Start Server:**
```bash
cd server
npm start
```

**Run Tests:**
```bash
python test_api_endpoints.py
```

### Essential Files

**Configuration:**
- `server/.env` - Database credentials (PROTECTED)
- `.gitignore` - Ignored files

**Database:**
- `server/consolidated_database.sql` - Full database (9.9 MB)

**Code:**
- `server/server.js` - API server (all 19 endpoints)

**Tests:**
- `test_api_endpoints.py` - Automated test suite
